Feigenbaum constant

Computing Feigenbaum constant using SymPy

So I saw this on Stephen Wolfram's blog where a simple recursive equation can yield chaotic behaviour and seems to have properties like Hopf bifurcation. I wanted to check out myself.

$$ x_{i+1} = a x_i (1 - x_i) $$

Reference: https://blog.stephenwolfram.com/2019/07/mitchell-feigenbaum-1944-2019-4-66920160910299067185320382/

from functools import lru_cache

@lru_cache(maxsize=8)
def f(i, a, x0):
    if i == 0:
        return x0
    else:
        return a * f(i-1, a, x0) * (1 - f(i-1, a, x0))

from statistics import mean

def series_mean(n, a, x0=1/3):
    s = series(n, a, x0)
    return s, mean(s[n//3:])

Bifurcation diagram¶

In the blog the following Wolfram code is used to generate the calculate the bifurcation. The first 50 values of the series are ignored to avoid transients and upto 300 values are calculated vfor each value of $a$.

ListPlot[Flatten[
  Table[{a, #} & /@ 
    Drop[NestList[Compile[x, a x (1 - x)], N[1/3], 300], 50], {a, 0, 
    4, .01}], 1], Frame -> True, FrameLabel -> {"a", "x"}]

Before we do that, let us see if it makes any difference if we vary the initial condition.

The average value where the series oscillates around does not seem to depend on the value of $x_0$. So now instead of plotting the mean, we can plot the full distribution where of values where the series oscillates around.

In [46]:
import numpy as np
N = 300
a_values =  np.linspace(0, 4, 100)

plt.figure()
for a in a_values:
    s = series(N, a)[50:]
    plt.scatter(a * np.ones_like(s), s, s=1, c="r")
    plt.xlabel("a")
    plt.ylabel("series distribution")

There is our multiple Hopf bifurcation :) Let us see what happens for $a> 3.5$.
